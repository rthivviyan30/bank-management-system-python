# 🏦 Bank Management System (Python)

## 📌 Description
A simple command-line banking system using Python OOP.

## ✨ Features
- Deposit money
- Withdraw money with validation
- Check balance
- Transaction history

## 🛠️ Technologies Used
- Python

## 🚀 How to Run
```bash
python main.py
```

```

1. Deposit
2. Withdraw
3. Check Balance
4. Transaction History
5. Exit
```
